# Student AI Chatbot (Groq API)

A simple AI chatbot built using Python and Streamlit, powered by the Groq API. It answers academic and programming-related questions.

## Features

* Chatbot interface using Streamlit
* AI-generated responses using Groq API
* Handles basic student queries
* Optional chat history storage

## Tech Stack

* Python
* Streamlit
* Groq API

## Project Structure

```
student-ai-chatbot/
│── app.py
│── chatbot.py
│── requirements.txt
│── README.md
```

## Installation

1. Clone the repository

```
git clone https://github.com/your-username/student-ai-chatbot.git
cd student-ai-chatbot
```

2. Install dependencies

```
pip install -r requirements.txt
```

3. Set Groq API key

Windows:

```
set GROQ_API_KEY=your_api_key
```

Mac/Linux:

```
export GROQ_API_KEY=your_api_key
```

4. Run the app

```
streamlit run app.py
```

## Usage

Enter a question in the input box and get an AI-generated response.
